== Installation ==
= Using The WordPress Dashboard =
* Navigate to the 'Add New' in the plugins dashboard
* Search for Eclerx Newsletter
* Click Install Now
* Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =
* Navigate to the 'Add New' in the plugins dashboard
* Navigate to the 'Upload' area
* Select eclerx-newsletter.zip from your computer
* Click 'Install Now'
* Activate the plugin in the Plugin dashboard

= Using FTP =
* Download eclerx-newsletter.zip
* Extract the eclerx-newsletter directory to your computer
* Upload the eclerx-newsletter directory to the /wp-content/plugins/ directory.
* Activate the plugin in the Plugin dashboard
